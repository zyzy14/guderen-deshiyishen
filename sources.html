古籍里的中国味道

— 从《齐民要术》到《山家清供》· 一本写给厨房的千年食典 —

中国人对"吃"的讲究，早在两千年前的简牍纸卷里，就已落笔成章。

本页带你速览「古人的深夜食堂」背后的几部关键典籍，与几位爱吃的文人。

项目里那四十余道食谱，并非凭空杜撰，而是逐道考据自农书、食单、药典与文人笔记——从北魏《齐民要术》到清人《粥谱》《闲情偶寄》，由宋人《山家清供》《东京梦华录》及苏轼、陆游诸家。读懂这些书，你报出的每一样家常食材，背后都站着一段实实在在的饮食传统。

群书 · 食事之源

齐民要术
作者 贾思勰 · 北魏（约 533—544 年）
现存最早、最完整的农学巨著，被誉为"中国古代农业百科全书"。全书十卷九十二篇，卷七至卷九专讲食品加工——炙、脯、酱、菹、羹、粥、饼、醴酪，乃至酿酒造醋，皆有细法。今人复刻的"胡炮肉""蒸缹鸡"即出于此。
本店复刻 → 胡炮肉 杏酪粥 蒸缹鸡 熟菘
随园食单
作者 袁枚 · 清（乾隆五十七年 / 1792 刊行）
中国历史上第一部真正成体系的"文人菜谱"。分须知单、戒单、海鲜单、江鲜单、特牲单、羽族单、水族单、杂素菜单、饭粥单、茶酒单等十四单，记菜三百余道。袁枚有"耳餐""目食"之戒——反对铺张，主张适口者为珍。
本店复刻 → 蒋侍郎豆腐 醉虾 捶鸡 · 鸡粥
本草纲目
作者 李时珍 · 明（1578 成书 · 1596 金陵初刊）
古代药物学集大成之作，亦是一部"食疗"宝典。五十二卷、收药近一千九百种，谷部、菜部、果部、兽部、禽部、鳞介部中无数可食之物，皆详其性味、主治。今人复刻的诸般药粥，皆本其"药食同源"之旨。
本店复刻 → 百合粥 山药粥 枸杞粥 莲子羹
山家清供
作者 林洪 · 南宋（约 13 世纪）
文人清供食谱之冠，一菜一名、一诗一典。记雪天拨霞供（火锅之雏形）、蟹酿橙、碧涧羹、傍林鲜等山林风味，把"吃"写成风雅。与李渔"蔬食第一"一脉相承，是本项目素食与清供菜式的重要来源。
本店复刻 → 拨霞供 蟹酿橙 金玉羹 傍林鲜
粥谱
作者 曹庭栋 · 清（乾隆年间）
一部专讲粥的食书，收粥方百余，以粥养身。分谷类、蔬类、果类、药类，详其性味主治。芡实、苏子、杏仁、胡桃、菊花诸粥，皆本其法——深夜一瓯，最是寻常人家的温润慰藉。
本店复刻 → 芡实粥 苏子粥 杏仁粥 菊花粥
云林堂饮食制度集
作者 倪瓒 · 元（1301—1374）
元四家之一倪瓒的私房食单，尚清鲜、重本味。所记"云林鹅"制法精到，后世奉为名馔；烧萝卜诸式，皆文人治庖的雅洁范本。
本店复刻 → 云林鹅 烧萝卜
东京梦华录
作者 孟元老 · 南宋（追忆北宋汴京）
追忆徽宗朝汴京繁华，市井夜市的饮食百态尽收笔底：旋炙猪皮肉、洗手蟹、灌肺、爊物……街边即炙即食的烟火气，正是"深夜食堂"最古早的模样。
本店复刻 → 洗手蟹 旋炙猪肉

文人食事 · 老饕列传

苏

苏轼

北宋 · 1037—1101 · 号东坡居士，自号"老饕"

贬谪黄州时写《猪肉颂》："净洗铛，少著水，柴头罨烟焰不起。待他自熟莫催他，火候足时他自美。"——东坡肉由此得名，是把困顿吃成风流的范本。

又创东坡羹，以蔓菁、芦菔（萝卜）煮而不用鱼肉，清贫中见真味；另作《老饕赋》《菜羹赋》，谈吃谈得理直气壮。一句"无竹令人俗，无肉使人瘦"，道尽文人的饮食脾性。

本店复刻 → 东坡肉 东坡羹 熟菘
张

张翰

西晋 · 字季鹰 · 一句"莼鲈之思"成千古归隐典故

据《世说新语·识鉴》：张翰在洛阳为官，秋风起，忽念吴中菰菜、莼羹、鲈鱼脍，叹曰：

"人生贵得适意尔，何能羁宦数千里以要名爵！"

遂命驾归乡。一碗莼鲈羹，吃的是故乡与自由——这是中国文人辞官史上最"好吃"的理由。

本店复刻 → 莼鲈羹
渔

李渔

清 · 1611—1680 · 字笠翁，《闲情偶寄》写尽生活雅趣

《闲情偶寄·饮馔部》开宗明义"蔬食第一"——于肉食之上，独推笋、蕈、莼、菜，谓笋为"蔬食中之领袖"，且"从来至美之物，皆利于孤行"。

他不信"有味者使之出，无味者使之入"的俗套，宁可白水一煮，存其本鲜。一份白水笋、一碟醋拌萝卜丝，便是文人治庖的清通。

本店复刻 → 白水笋 素蕈 醋拌萝卜丝
陆

陆游

南宋 · 1125—1210 · 号放翁，诗万首，亦好谈食养

放翁晚年尤重食养，作《食粥》诗：

"世人个个学长年，不悟长年在目前。我得宛丘平易法，只将食粥致神仙。"

一碗淡粥，在他笔下成了养生大道。又爱笋、爱荠、爱素羹——与苏轼、李渔同是"以吃写心"的文人。食粥一味，最宜深夜独啜。

考据小笺 · 冷知识

"食单"二字 袁枚用得最妙

不是菜谱，是"菜单"——一本可以慢慢翻的闲书。袁枚写它，意在"知味"而非炫技。

古代版"叫花鸡" 胡炮肉

《齐民要术》载：把羊肉馅塞进羊肚，埋进烧热的羊粪火里煨熟。是南北朝的野趣炙法。

李时珍论粥 食疗之良

称"粥极柔腻，与肠胃相得，最为饮食之良"——一碗药粥，养的是寻常人家的身子。

最"好吃"的辞职 张翰

为口莼菜鲈鱼弃官归乡，成就成语"莼鲈之思"。乡愁，原来是可以下筷的。

古人的夜市 东京梦华录

南宋人追忆汴京，夜市"直至三更尽，才五更又复开张"——旋炙猪肉、洗手蟹就着灯火即炙即食，正是千年前的深夜食堂。

文人的鹅 云林鹅

画家倪瓒好洁亦好吃，所记烧鹅法精到，后世名之"云林鹅"。一盏清供，连荤腥也吃得雅。

已入食单 群贤毕至

李渔《闲情偶寄》、林洪《山家清供》、曹庭栋《粥谱》、倪瓒《云林堂》、孟元老《东京梦华录》及陆游诗，皆已收入本店知识库，可逐一复刻。

半文半白 今人演绎

本店引文皆有出处，而"深夜独食者"的那封短信，是今人为古法添的体温。
